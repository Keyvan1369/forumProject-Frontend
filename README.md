# Forum Project Frontend

A modern and responsive forum application frontend built with React, Vite, and Tailwind CSS. This project provides a clean user interface for users to browse discussions, create posts, interact with topics, and engage with the community through a fast and intuitive experience.

---

##  Tech Stack

* React 19
* Vite
* React Router 7
* Tailwind CSS 4
* Axios
* JavaScript (ES6+)
* ESLint

---

##  Features

* Modern and Responsive UI
* Fast Development with Vite
* Client-Side Routing
* API Integration with Axios
* Community Discussion Interface
* User-Friendly Navigation
* Reusable React Components
* Mobile-Friendly Design
* Scalable Frontend Architecture

##  Prerequisites

Before getting started, make sure you have the following installed:

* Node.js (v18 or higher recommended)
* npm

---

##  Installation

Clone the repository:

```bash
git clone https://github.com/Keyvan1369/forumProject-Frontend.git
```

Navigate to the project directory:

```bash
cd forumProject-Frontend
```

Install dependencies:

```bash
npm install
```

---

##  Environment Variables

Create a `.env` file in the project root if your application requires API configuration:

```env
VITE_API_URL=http://localhost:5000/api
```

Replace the URL with your backend API endpoint when deploying.

---

##  Running the Application

Start the development server:

```bash
npm run dev
```

The application will be available at:

```text
http://localhost:5173
```

---

##  Build for Production

Generate an optimized production build:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```
